# eks-course

AI-assisted learning workspace for Amazon EKS, structured using the
[teach](https://github.com/mattpocock/skills/blob/main/skills/productivity/teach/SKILL.md)
skill.

The goal is to learn EKS well enough to create clusters, configure node groups
with custom AMIs, and validate that bootc-based nodes work correctly — see
[MISSION.md](MISSION.md) for details.

## Structure

| Path | Description |
|------|-------------|
| [MISSION.md](MISSION.md) | Learning goals, constraints, and scope |
| [RESOURCES.md](RESOURCES.md) | Curated reference materials |
| [NOTES.md](NOTES.md) | Teaching notes and learner context |
| `lessons/` | Interactive HTML lessons |
| `learning-records/` | Incremental knowledge snapshots |
| `reference/` | Quick-reference HTML docs |
